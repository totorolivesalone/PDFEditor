# PDF Editor

PDF Editor toolbox containing scripts to manipulate pdfs natively because I don't want to pay for yet another subscription :Ãž  

## Pre-requisites
Python 3.9 or higher ;
pip or conda package manager

## Installation
### Option 1: Using Conda
``` bash
git clone https://github.com/totorolivesalone/PDFEditor.git
cd PDFEditor
conda env create -f environment.yml
conda activate PDFEditor
```
### Option 2:Using pip
``` bash
git clone https://github.com/totorolivesalone/PDFEditor.git
cd PDFEditor
pip install -r PyPDF2
```
